Refcode

Refcode is a curriculum to introduce beginners to coding. It is put on by the Refugee Career Hub as part of their computer literacy program. After going through this curriculum, students will learn the basics of how to build a website, and students will know a little bit about what working as a professional developer is like.

Students will learn to use the tools that professional developers use to create websites. In this course, students will be introduced to Atom (an IDE or code editor), Chrome Inspect tools, GitHub Desktop, Cyberduck (an FTP client), and Freenom.com (a domain registrar).

In this class we will learn about the strategies, workflows, and tools that professionals use to stay organized, communicate, and manage their time. We will learn about Agile Methodology, stand-up meetings, sprint cycles, kanban boards (with Trello), Slack chat, GitHub, and pair-programming. Each student will work towards creating a personal website by the end of the class. The student will register their own domain name and upload it to a web hosting server (provided by Refugee Career Hub).

Nature

Coding Classes

The Refcode Level 1 class gives refugees and immigrants a glimpse of what a career in software engineering would look like. In addition to teaching students to build and code a website, students learn to use industry tools like GitHub, Slack, an IDE, and browser dev tools. They also learn important strategies mindsets and soft skills such as how to solve problems, how to use online references and resources, where to go for help, how to ask questions.

Mentorship

Completing the class is just the beginning. Graduates interested in pursuing a career are matched with a mentor experienced in the field who can provide further guidance and direction for next steps. Whether that means applying to a school, studying a free online curriculum, building projects at home, or interviewing for that first developer job, mentors are prepared to give encouragement and leverage their own networks and experience on behalf of their mentees.

TeamCode

It has been said that 70% of growth and development comes from on-the-job experience, 20% comes from collaboration, and only 10% comes from traditional education. If that is true, then this program gets participants to 90%. Building a website on your own is challenging, but contributing to a team project is exponentially more difficult. The core of TeamCode is collaboration and communication. Graduates who join as contributors hone these skills by building a real-world project together as part of an agile team using GitHub, a project board, and Slack. Contributors define and scope the work and assign tasks during sprint planning meetings and simultaneously assemble a working website following principles of Atomic Design.
